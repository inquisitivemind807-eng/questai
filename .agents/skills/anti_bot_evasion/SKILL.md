# Instructions

To evade bot detection systems on sites like Indeed, you MUST perform the following actions:
- Add random delays between actions (1–3 seconds).
- Move the mouse naturally across the screen before making clicks.
- Scroll the page slowly, simulating human reading speed.
- Avoid repeating the exact same pattern of movements or timing.

This is critical because Indeed and similar platforms detect bots aggressively.
