---
name: Form Detection
description: Scans and identifies form input fields before attempting submission.
---

# Instructions

When on an application form:
1. First, scan the entire page and identify all input types:
   - Text fields
   - Dropdowns
   - Radio buttons
   - Checkboxes
   - File uploads
   - Multi-step pagination

2. Do not attempt to submit the form until all required fields have been correctly filled out or interacted with.
